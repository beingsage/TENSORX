## HOW TO INTEGRATE YOUR TRAINED MODELS

### Option 1: XGBoost
```typescript
import xgboost from 'xgboost'; // or another library
import { predictValuation } from '@/lib/models/valuation';

// In /lib/models/valuation.ts, replace:
export function predictValuation(features) {
  const booster = xgboost.load('./trained-gbm.bin');
  const prediction = booster.predict(features.tabularFeatures);
  return { pointEstimate: prediction, ... };
}
```

### Option 2: TensorFlow
```typescript
import * as tf from '@tensorflow/tfjs';

export async function predictValuation(features) {
  const model = await tf.loadLayersModel('file://./model/model.json');
  const tensor = tf.tensor2d([Object.values(features.tabularFeatures)]);
  const prediction = model.predict(tensor);
  return { pointEstimate: prediction.dataSync()[0], ... };
}
```

### Option 3: Python (FastAPI Wrapper)
```python
# Python service
from fastapi import FastAPI
from xgboost import XGBRegressor
app = FastAPI()

model = XGBRegressor()
model.load_model('model.json')

@app.post("/predict")
async def predict(features):
    return {"valuation": model.predict(features)}
```

Then call from Node.js:
```typescript
const response = await fetch('http://python-service:8000/predict', {
  method: 'POST',
  body: JSON.stringify(engineeredFeatures)
});
```
